# Sistema de Controle de Estacionamento

Sistema de gerenciamento de estacionamento para controle de entradas, saídas, cobrança por hora e geração de relatórios. 
Ideal para pequenos e médios estacionamentos que precisam automatizar o controle de vagas, veículos e clientes.

## 🧰 Funcionalidades principais

- Cadastro de clientes (nome, CPF/CNPJ, telefone, etc.).  
- Cadastro de veículos (placa, modelo, cor, cliente associado).  
- Controle de entrada e saída de veículos com registro de horário.  
- Cálculo automático do valor a ser cobrado com base em taxa por hora/diária.  
- Visualização de vagas ocupadas e vagas livres em tempo real.  
- Relatórios de receita por período, histórico de veículos e fluxo de entrada/saída.  
- Interface fácil de usar (web ou desktop, conforme sua implementação).
